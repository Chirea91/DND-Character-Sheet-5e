# DnD Character Sheet (Python)

Lernprojekt zur Vertiefung meiner Python-Kenntnisse.

Das Projekt basiert auf den Regeln von Dungeons & Dragons 5e und dient dazu, 

typische Programmierkonzepte praktisch anzuwenden, darunter:

      - Funktionen
      - Datenstrukturen
      - JSON-Dateiverarbeitung
      - Objektorientierte Programmierung (OOP)
      - Fehlerbehandlung
      - Projektstrukturierung
      - Versionsverwaltung mit Git/GitHub

## Aktueller Funktionsumfang

### Charakterverwaltung

    - Attribute bearbeiten
    - Charakterdaten anzeigen
    - Charakterdaten speichern und laden (JSON)

### Würfelsystem

    - Allgemeine Würfelwürfe
    - Saving Throws
    - Skill Checks
    - Automatische Berechnung und Anwendung von Attributsmodifikatoren nach D&D-5e-Regeln

### Inventarverwaltung

    - Gegenstände hinzufügen
    - Gegenstände verwenden
    - Gegenstände verkaufen
    - Gegenstände entfernen

### Währungssystem

    - Automatische Umrechnung und Ausgabe der verschiedenen Geldwerte

### Menüsystem

    - Navigation durch verschiedene Programmfunktionen

## Geplante Funktionen


### Projektstruktur 

    - Bereiche Auslagern, um Chaos in der Struktur zu verringern

### Kampfsystem

    - Trefferermittlung
    - Schadensberechnung
    - Advantage / Disadvantage
    - Kritische Treffer
    - Kritische Fehlschläge
    - Initiative
    - Trefferpunkte (HP)
    - Armor Class (AC)

### Magiesystem

    - Zauberliste
    - Verwaltung von Spell Slots
    - Spell Attack Bonus
    - Spell Save DC

### Simulation und Analyse

    - Würfelsimulationen
    - Statistische Auswertungen
    - Durchschnittswerte
    - Median
    - Speicherung von Simulationsergebnissen
    - Vorbereitung für spätere Datenanalysen

## Projektstatus

Aktives Lernprojekt.

Der Fokus liegt aktuell auf dem Ausbau der Python-Kenntnisse und einer sauberen Projektstruktur.

Langfristiges Ziel ist es, das Projekt so weit auszubauen, 

dass eine komplette D&D-Session damit unterstützt werden kann.
